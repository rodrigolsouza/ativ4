COMPARAÇÃO ENTRE AS PRINCIPAIS LINGUAGENS QUE SERÃO ESTUDADAS

Linguagem Tipo conversão para ling. máquina Paradigma de programação suportados Tipagens existentes Grau de abstração Opinião do Estudante
PHP Interpretada Procedural, reflexiva, orientada a objetos e funcional Dinâmica/Fraca e Gradual Alto nível Interessante para desenvolvimento web e bem eclética quanto tipagem e paradigmas
C# Compilada Estruturada,imperativa,concorrente,funcional,genérica,orientada a eventos,orientada a objetos e reflexiva Estática/forte dinâmica/forte Alto nível É uma linguagem muito interessante de se aprender já que a Microsoft ainda domina o mercado com seus produtos.
Java Híbrida Orientada a objetos, estruturado, imperativo, genérico, funcional, reflexivo e concorrente Estática/forte Alto Nível Linguagem com curva de aprendizado muito longa, tem uam comunidade muito boa e é considarada uma das mais seguras para desenvolvimento web e mobile.
Javascript Interpretada Orientada a objetos, protótopos, imperativo e funcional Dinâmica/fraca Alto nível Possui recursos e compatibilidades muito bons. Com a expansão para o lado do servidor e programação backend passou a ser fundamental se dominar.
Python Interpretada Orientada a objetos, imperativo e funcinal Dinâmica/forte, Duck e gradual Alto nível Curva de aprendizagem muito boa, recursos muito bons, está sendo aperfeiçoada para a área de ciência de dados e está ficando muito poderosa nesse caminho.
R Interpretada Sequencialização, orientado a objetos, imperativo, dinâmico, processual, recíproc0 Dinâmica/fraca Alto nível Possui uma alta flexibilidade e potência para quem trabalha com dados e é acessível para não programadores, mas deixa a desejar em eficiência e velocidade e não é para desenvolvimento web logo ela é bem restrita para a área de big Data e afins.
